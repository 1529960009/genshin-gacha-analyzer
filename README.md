# Genshin Gacha Analyzer | 原神抽卡记录分析工具

## 如何获取xlsx文件？
sunfkny的[抽卡记录导出工具](https://ngabbs.com/read.php?tid=25004616)  
lvlvl的[electron版抽卡记录导出工具](https://ngabbs.com/read.php?tid=25559039)，导出数据后选择导出Excel
